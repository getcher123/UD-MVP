Верни JSON-объект со структурой:
{
  "title": string|null,
  "subtitle": string|null,
  "blocks": [
    {
      "type": "title"|"subtitle"|"paragraph"|"list",
      "text": string
    }
  ],
  "points": [string],
  "contacts": {
    "address": string|null,
    "website": string|null,
    "phone": [string],
    "messengers": [string]
  },
  "amenities": [string],
  "unknown": [string]
}
Требования:
- Строго валидный JSON.
- Внутренние кавычки экранируй \\"\\".
- Пустые списки допускаются.
